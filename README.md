# Fizyoterapi.com
# Java Spring MVC Physiotherapy Clinic Appointment Making App




## Technologies

---
- Java 8
- Spring Boot 2.7.10
- Spring Data JPA
- MySql
- JSP
- Bootstrap 5
- Maven
- Junit
- Mockito
